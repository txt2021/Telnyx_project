# Telnyx_project
Project for testing Telnyx website: https://telnyx.com/

10 test cases for sign in, sign up, navigation, reset password and resend emal areas

## Requirements
cypress: ^10.6.0

Node.js : 14^

## Install
git clone https://github.com/txt2021/Telnyx_project.git

npm install 

## Run
npx cypress run

## Open
npx cypress open
